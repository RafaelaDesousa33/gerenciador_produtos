Gerenciador de Produtos


O Gerenciador de Produtos é um sistema desenvolvido em Python para facilitar o gerenciamento de produtos em pequenos negócios ou projetos pessoais.
Ele permite cadastrar, listar, deletar e aplicar descontos em produtos, utilizando conceitos de Programação Orientada a Objetos (POO) para uma estrutura organizada e modular.

Funcionalidades

Cadastro de novos produtos, incluindo nome, preço e categoria.

Listagem de todos os produtos cadastrados.

Remoção de produtos existentes.

Aplicação de descontos nos produtos.

Interface simples baseada em terminal para interação com o usuário.

Tecnologias Utilizadas

Python 3.x

Programação Orientada a Objetos (POO)

Estrutura do Projeto

O projeto está organizado em módulos para melhor manutenção:

Produtos.py: define a classe Produtos e suas propriedades.

Gerenciamento_produtos.py: gerencia operações de produtos (adicionar, remover, listar, aplicar desconto).

app.py: ponto de entrada do sistema, responsável por interagir com o usuário.

Como Executar

Certifique-se de ter o Python 3.x instalado em sua máquina.

Clone o repositório:

git clone https://github.com/RafaelaDesousa33/gerenciador_produtos.git


Acesse a pasta do projeto:

cd gerenciador_produtos


Execute o arquivo principal:

python app.py


Siga as instruções na tela para gerenciar seus produtos.
